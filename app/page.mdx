Oracle Fusion Sales and Service Cloud is part of **Oracle Fusion Cloud Applications**.  
It helps organizations manage **customers, sales activities, and customer support** using a cloud-based system.

This page provides a **high-level understanding** of how the platform works and what it is used for.

---

## What is Oracle Fusion Cloud (OFC)?

**Oracle Fusion Cloud (OFC)** is a **cloud-based enterprise platform** provided by Oracle to manage business processes.

It runs completely in a **web browser** and is maintained by Oracle, including upgrades and security.

### What is it used for?

Oracle Fusion Cloud is used to:

- Manage customer data
- Sell products and services
- Handle customer issues
- Automate business processes

## Sales Cloud

### What does Sales Cloud do?

Sales Cloud helps sales teams:

- Track potential customers
- Manage customer accounts
- Follow sales opportunities

## Objects in Oracle Fusion

An **Object** is a place where business data is stored.

You can think of it as:
- A table in a database
- A business entity in the application

Objects store data such as customers, orders, or service requests.

### Object parts

Each object can include:
- Fields (data values)
- Records (rows of data)
- Pages (UI screens)
- Business rules
- Security settings

## Customization Using Groovy Script

Oracle Fusion allows limited scripting using **Groovy**.

Groovy is used to add **business logic**, not to build full applications.

### What can Groovy do?

- Validate data
- Auto-update fields
- Control behavior during save or update
- Perform calculations

Groovy always runs **inside Oracle Fusion**.

### Where Groovy is used

| Area | Purpose |
|----|----|
| Field level | Validation |
| Object level | Triggers |
| Global level | Reusable functions |

## Configuration vs Customization

Oracle Fusion supports two approaches:

### Configuration
- Done using UI tools
- No code required
- Used for layouts, security, fields

### Customization
- Uses Groovy scripts
- Used for rules and automation
- Controlled and limited by Oracle
